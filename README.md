# 🌍 NLP Toolkit

A complete Natural Language Processing (NLP) toolkit that performs:

- 🌐 Language Detection  
- 📝 Text Summarization  
- 🔤 Language Translation  

---

## 🚀 Features

- Detects language with **high accuracy (~99%)**
- Generates summaries using **Transformer models**
- Translates text between multiple languages
- Clean and modular code (separate scripts)

---

## 🧠 Technologies Used

- Python  
- Scikit-learn (SVM, TF-IDF)  
- TensorFlow / Keras  
- Hugging Face Transformers  
- Matplotlib & Seaborn  

---

## 📊 Model Details

### 🔹 Language Detection
- Model: **Linear SVM**
- Feature Extraction: **Character-level TF-IDF**
- Accuracy: **99.18% 🔥**
